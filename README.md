# Agentic AI Full-Stack Portfolio

Professional single-page GitHub Pages portfolio focused on software engineering and agentic AI full-stack work.

## Files

- `index.html` - Page structure and content
- `styles.css` - Theme, layout, responsive design, animations
- `script.js` - Scroll reveal and active nav behavior

## Customize Content

Open `index.html` and replace placeholder values:

- `[live URL]`
- `[repo URL]`
- `[pdf URL]`
- `[image URL]`
- `[URL]` in other project/contact links
- `[email]`

## Run Locally

Open `index.html` in your browser, or use any simple local server.

## Deploy to GitHub Pages

1. Push this repository to GitHub.
2. Go to repository `Settings` -> `Pages`.
3. Set `Source` to `Deploy from a branch`.
4. Select branch `main` (or your default branch) and folder `/ (root)`.
5. Save and wait for the published site URL.
