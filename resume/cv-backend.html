Autonomous Agentic Trader

Live: manager-trader.agentairg.site
Project Overview

A multi-agent trading simulation that replaces "black-box" AI decision-making with a four-gate portfolio-control pipeline. The system orchestrates four distinct LLM personas—Warren (Conservative), George (Aggressive), Ray (Balanced), and Cathie (Growth)—each operating under a global risk policy that enforces 14+ deterministic rules to ensure capital preservation and auditability.

Key Technical Details
  • Deterministic Risk Gatekeeping: Implements a rigorous validation layer that evaluates every trade proposal against real-time portfolio snapshots, enforcing constraints like a 10% drawdown block, re-entry cooldowns, and position exposure limits.
  • MCP Isolation Contract: Utilizes Model Context Protocol (MCP) boundaries to ensure agents never have direct brokerage access. Instead, they interact with typed tool contracts for accounts, multi-source market pricing (Polygon/Alpaca/Brave), and per-trader PostgreSQL memory graphs.
  • SSE-First Real-Time Architecture: Features a resilient dashboard powered by Server-Sent Events (SSE) with a 300-event ring buffer. This allows the UI to handle disconnections gracefully via Last-Event-ID replay and automated REST resyncing, maintaining a "live" state even during engine restarts.
  • Automated Cloud Infrastructure: Managed via Terraform and deployed on AWS EC2 using Docker and GitHub Actions. The stack includes a dedicated read-model layer to ensure dashboard consistency regardless of whether the trading engine is active or offline.
