MediNotes — Agentic Clinical Documentation Platform

Agentic healthcare SaaS that converts multimodal visit inputs into structured, verifiable summaries with evidence mapping, SSE streaming + resumable jobs, and a scoped clinical chat assistant bounded to each patient's visit summaries and consultation context.

Highlights

  • Multimodal intake: notes + docs/PDF + audio + images (OCR/transcription)
  • Streaming output via SSE with resumable job events (reconnect support)
  • Hub-and-spoke orchestration: summary orchestrator + specialized agents (extraction, research via MCP, critic, evidence mapping, memory/RAG, email)
  • Critic QA loop + regeneration to reduce hallucinations and missed facts
  • Evidence-linked summaries mapped back to source snippets for auditability
  • Scoped chat assistant: context-aware Q&A restricted to each patient's visit summaries + current consultation (no out-of-context answers)
  • Delivery: email dispatch via Resend; persistence via vector store (RAG memory) + Redis job store

Tech Stack

FastAPI Python Next.js/React TypeScript SSE Vector Store (RAG) Upstash Redis MCP (Brave Search) Resend Clerk Docker AWS App Runner/ECR
