Automation Operations

Recent private-client ecommerce operations automation work organized around two projects: Local Pickup Automation and Warehouse-to-Shopify Inventory Automation. Client details are intentionally omitted. The work converted manual warehouse, storefront, customer-notification, and inventory steps into repeatable workflows using the tools the operations team already used.

Local Pickup Automation

Manual operation: warehouse and customer-service staff had to check local pickup orders, identify when an order was ready, update the storefront state, send customer emails, track reminders, and stop follow-ups after pickup, fulfillment, or cancellation.

Automated operation: ShipStation stays the warehouse trigger, Make.com moves pickup-ready state into Shopify, and Shopify Flow sends ready-for-pickup emails, reminder emails, cancellation/refund messaging, and pickup-completed confirmation.

Reliability controls: explicit Shopify tags track processed states such as ready email sent, 5-day reminder, 8-day reminder, cancelled, and completed. Fulfillment and cancellation state stop later workflow steps so customers do not receive duplicate or incorrect messages.

Warehouse-to-Shopify Inventory Automation

Manual operation: warehouse staff had to scan orders and SKUs, capture SOP photos, mark completion, and coordinate inventory updates for non-Shopify marketplace orders. The risk was missed updates, unclear status, and double inventory deduction for orders already handled through Shopify.

Automated operation: AppSheet provides the warehouse scanning interface on mobile, backed by Google Sheets. Apps Script and the Shopify Admin API create the bridge from completed warehouse scans into Shopify, so existing inventory-system integrations can process the order flow.

Reliability controls: source detection and skip rules prevent duplicate deduction for existing Shopify orders. Operator-visible sync statuses, source-order IDs, timestamps, skipped/failed states, and error write-back make handoff and troubleshooting clearer.
